PVE host use dhcp

2021-10-30 1 min read # Network # PVE # Linux

编辑文件

/etc/network/interfaces

auto lo
iface lo inet loopback

iface ens33 inet manual

auto vmbr0
iface vmbr0 inet dhcp
    bridge_ports ens33
    bridge_stp off
    bridge_fd 0

其中 ens33 可能是其他名称 为物理端口名

应用更改

reboot
or
ifdown vmbr0 && ifup vmbr0
