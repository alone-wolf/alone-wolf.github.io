Ubuntu 增加swap

2021-01-27 1 min read # Ubuntu # Shell # Linux

创建swap文件

sudo dd if=/dev/zero of=swapfile6 bs=1024 count=6000000
sudo mkswap -f swapfile6

启用swap文件

sudo swapon swapfile6
使用free -m查看内存

停用swap文件

sudo swapoff swapfile6

自动挂载swap文件

将如下内容添加到 /etc/fstab 文件中
/swapfile6 /swap swap defaults 0 0
